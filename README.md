# MFACE-A-high-resolution-model-of-field-aligned-currents
MFACE is a high-resolution field-aligned currents model derived from ten years of CHAMP data with a novel technique based on EOF decomposition. Predictor variables include MLat, MLT, DoY, IMF conditions, Vsw and AE index.
